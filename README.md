# Low Price Villager

Make every villager trade cost just **1 Emerald** instantly in Minecraft Bedrock Edition.

This lightweight addon automatically applies the **Hero of the Village** effect without disabling achievements. It also includes an optional Enchant Aura and an easy in-game toggle menu.

## Features

- Every villager trade costs only **1 Emerald**
- Optional Enchant Aura
- Toggle features using a Stick
- Achievements remain enabled
- Lightweight Script API implementation
- No commands required

## Compatibility

- Minecraft Bedrock Edition 1.26+
- Script API

## How to Use

1. Join your world.
2. Hold a **Stick**.
3. Use the Stick to open the menu.
4. Toggle:
   - Villager Discount
   - Enchant Aura
