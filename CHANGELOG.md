# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-20

### Added

- Hero of the Village effect that reduces all villager trades to **1 Emerald**.
- In-game toggle menu using a Stick.
- Optional Enchant Aura.
- Automatic activation for first-time players.
- Achievement-friendly implementation using the Minecraft Script API.
- Lightweight and optimized behavior pack.
- Support for Minecraft Bedrock Edition **1.26+**.

### Changed

- Initial public release.
